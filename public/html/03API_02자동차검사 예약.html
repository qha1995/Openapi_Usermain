자동차검사 예약

API 개요

자동차검사 예약 서비스는 국민이 자주 사용하는 민간 플랫폼을 통해 자동차검사 예약, 예약 조회·변경·취소, 문진표 작성 등을 할 수 있게 해줍니다. 이러한 작업을 지원하는 별개의 상세 API들로 구성되어있으며, 이용기관은 서비스 이용 신청 시 발급받은 API키를 통하여 API를 호출합니다.
