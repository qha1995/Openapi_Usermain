API 개요

문화누리카드 발급신청 및 이용내역 조회 서비스는 국민이 자주 사용하는 민간 플랫폼을 통해 문화누리카드 신규 발급신청, 재발급, 충전, 사용 등록, 잔액 확인 등을 할 수 있게 해줍니다. 이러한 작업을 지원하는 별개의 상세 API들로 구성되어있으며, 이용기관은 서비스 이용 신청 시 발급받은 API키를 통하여 API를 호출합니다.
